linearDependencies

src/linearDependencies.js

Creates a matrix which represents the dependencies between rows. If a row is a linear combination of others rows, the result will be a row with the coefficients of this combination. For example : for A = [[2, 0, 0, 1], [0, 1, 6, 0], [0, 3, 0, 1], [0, 0, 1, 0], [0, 1, 2, 0]], the result will be [[0, 0, 0, 0, 0], [0, 0, 0, 4, 1], [0, 0, 0, 0, 0], [0, 0.25, 0, 0, -0.25], [0, 1, 0, -4, 0]]

linearDependencies(matrix: Matrix, options: Object?): Matrix
Parameters
matrix (Matrix)
options (Object? = {}) includes thresholdValue and thresholdError.
Name Description
options.thresholdValue number (default 10e-10) If an absolute value is inferior to this threshold, it will equals zero.
options.thresholdError number (default 10e-10) If the error is inferior to that threshold, the linear combination found is accepted and the row is dependent from other rows.
Returns
Matrix: the matrix which represents the dependencies between rows.
